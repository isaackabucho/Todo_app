# A Todo-List App
This project is a simple Todo app built with Laravel and Livewire for users to manage their tasks.




## Language
-Laravel 11
-Livewire 3

## How to setup and run
-Install Laravel and necesary dependencies

-run 'php artisan serve'

run 'npm run build'

run 'npm run dev'
